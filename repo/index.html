Tutone
Tutone is a theme. Actually, it's multiple themes. A flat theme and a gradient theme, each with over 140 icons. It also features themed status bar icons, themed music controls, several icon masks, themed settings icons, and more. Lot's more is on the way, including a boot logo, custom UI sounds, a Control Centre theme, and UI tweaks. Apply these with Anemone (recommended), or with WinterBoard + IconBundles.
